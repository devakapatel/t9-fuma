## **Aim:** Network Vulnerability Scanning using **OpenVAS**

### **What is OpenVAS?**
OpenVAS (Open Vulnerability Assessment System) is a **free, open-source scanner** for identifying:
- Misconfigurations
- Outdated software
- Weak passwords
- CVEs

Part of **Greenbone Vulnerability Management (GVM)**.
