## **Aim:** Install VMware Workstation on Ubuntu & Set Up Windows + Kali VMs

### **Introduction to Virtualization**
Run multiple OS on one machine using **VMware Workstation**.

---

### **Step-by-Step Procedure**

#### **Install VMware on Ubuntu**
```bash
cd ~/Downloads
chmod +x VMware-Player-*.bundle
sudo ./VMware-Player-*.bundle
```
> Launch: `vmplayer`

---

#### **Set Up Windows 11 VM**
1. Download ISO: [microsoft.com/windows11](https://www.microsoft.com/en-in/software-download/windows11)
2. New VM → Select ISO
3. Settings:
   - Name: `Windows11_VM`
   - RAM: **4096 MB**
   - Disk: **50 GB**
   - Network: **NAT**

---

#### **Set Up Kali Linux VM**
1. Download: [kali.org/get-kali](https://www.kali.org/get-kali/#kali-installer-images)
2. New VM → Select ISO
3. Settings:
   - Name: `Kali_Linux_VM`
   - RAM: **4096 MB**
   - Disk: **40 GB**
   - Network: **Bridged**

---
